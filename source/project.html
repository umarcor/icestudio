ICE format¶

Definition¶

  • Version: 1.2.

  • Package: project information.

  • Design: board information and circuit design.

  • Dependencies: all used dependencies in one level.

Extension: .ice

{
  "version": "1.2",
  "package": {
    "name": "",
    "version": "",
    "description": "",
    "author": "",
    "image": ""
  },
  "design": {
    "board": "",
    "graph": {
      "blocks": [],
      "wires": []
    }
  },
  "dependencies": {}
}

Block instances¶

{
  "id": "",
  "type": "",
  "data": {},
  "position": {
    "x": 0,
    "y": 0
  },
  "size": {
    "width": 0,
    "height": 0
  }
}
